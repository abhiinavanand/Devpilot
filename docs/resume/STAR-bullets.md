## STAR Examples
- **Situation**: MVP lacked observability and deployment rigor.
  **Task**: Design an enterprise-ready platform for DevOps + AI workflows.
  **Action**: Built microservices with API gateway, real-time updates, feature flags, Terraform IaC, and full monitoring stack.
  **Result**: Delivered a portfolio-grade SaaS demo with production-grade architecture and documentation.

- **Situation**: Manual incident response delayed recovery.
  **Task**: Establish incident response and monitoring practices.
  **Action**: Authored SLOs, runbooks, alert rules, Grafana dashboards, and incident response handbook.
  **Result**: Achieved clear operational playbooks and reduced mean time to mitigate in simulations.
