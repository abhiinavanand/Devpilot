# DevPilot AI Frontend

This is the portfolio-ready command center UI inspired by Linear, Jira, Vercel, Datadog, GitHub, and Grafana.

## Highlights
- Beautiful dashboards with advanced charts.
- Drag-and-drop Kanban workflow.
- Command palette (⌘K / Ctrl+K).
- AI assistant widget and live metrics.
- Terminal-inspired release panel.
- Dark/light mode toggle.

## Scripts
- `npm run dev`
- `npm run build`
- `npm run preview`
- `npm run storybook`

## Routes
- `/` Overview
- `/projects`
- `/deployments`
- `/incidents`
- `/slos`
- `/workbench`
